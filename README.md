# Not your Grandad's PHP

Team presentations on advances in PHP language over the last decade or two.

## Prerequisites

[`mise`](https://mise.jdx.dev/) used to install [presenterm](https://mfontanini.github.io/presenterm/). 

At the time of writing, PFA required self-compiling PHP 8.6 from [this PR](https://github.com/arnaud-lb/php-src/pull/22)
